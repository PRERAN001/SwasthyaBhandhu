SwasthyaBhandhu - AI Health Assistant Integration Guide
========================================================

OVERVIEW:
The AI Health Assistant has been integrated into the Patient Dashboard as a new tab.
It combines ElevenLabs conversational AI with Groq API for report generation.

FEATURES:
1. ElevenLabs Conversational AI widget for voice/text interaction
2. AI-powered health report generation using Groq API
3. Conversation summary input
4. Report storage and history
5. Download reports as text files

HOW IT WORKS:
1. Patient talks to the ElevenLabs AI agent about their health concerns
2. Patient summarizes the conversation in the text area
3. Click "Generate Health Report" to create a comprehensive medical report
4. Groq API (llama-3.3-70b-versatile model) analyzes the conversation
5. Report includes: Chief Complaints, Symptoms Analysis, Assessment, Recommendations
6. Reports are saved to localStorage and can be viewed/downloaded later
